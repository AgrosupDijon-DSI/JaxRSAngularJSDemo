JaxRSAngularJSDemo
=================

Demo of REST client-server using Java JaxRS and AngularJS.

Requirements :
* JDK >= 7,
* Eclipse IDE for Java EE Devs (Kepler),
* Glassfish 4.0,
* Recent web browser.
